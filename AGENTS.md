## Project Summary
Geetato Bakes & Bliss is a premium artisan bakery based in Indirapuram, Ghaziabad. The project is a full-stack website designed to position the brand as a top-tier B2B supplier for corporate clients and HORECA (Hotel, Restaurant, Cafe) establishments, while maintaining a strong D2C presence.

## Tech Stack
- **Framework:** Next.js 15 (App Router)
- **Styling:** Tailwind CSS 4
- **Icons:** Lucide React
- **Assets:** Supabase Storage (for logo and branding images)

## Architecture
- `src/app/(marketing)`: Contains all public-facing routes (Home, Products, B2B, About, etc.) with a shared layout.
- `src/components`: Shared UI components like Navbar and Footer.
- `src/app/globals.css`: Custom theme variables for Geetato branding (Hot Pink, Cream, Charcoal).

## User Preferences
- **Company Name:** Geetato
- **Tagline:** Fresh Bakes, Real Taste
- **Colors:** Hot Pink (#E91E63), Cream (#F5F1E8), Charcoal (#2C2C2C).
- **Aesthetic:** Professional, premium, modern, and bold.
- **Focus:** B2B supplier positioning with secondary D2C presence.

## Project Guidelines
- Use the provided Geetato logo and site icon.
- Primary CTA buttons should always be Hot Pink.
- Maintain high visual reality in images (reference provided).
- SEO optimized with proper meta tags and internal linking.

## Common Patterns
- Marketing pages use a shared layout in `src/app/(marketing)/layout.tsx`.
- Consistent use of cards and sections for service/product categorization.
